피닉스 기동
=====================
1) 시작 클릭 -> cmd -> 마우스 오른쪽 -> 관리자 모드 실행 또는 Visual Code에서 마우스 오른쪽 관리자 모드 실행
2) 명령어 창에서 아래 명령어 수행
```
mix phx.server
```

오류 해결
---------------------
### 관리자 모드 CMD실행(Window 11)
1)  시작 클릭 -> cmd -> 마우스 오른쪽 -> 관리자 모드 실행 또는 Visual Code에서 마우스 오른쪽 관리자 모드 실행
2)  명령어 창에서 아래 명령어 수행
```
mix deps.clean --all
mix deps.get
mix phx.server
```
