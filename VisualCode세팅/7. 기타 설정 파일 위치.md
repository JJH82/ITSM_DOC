아래의 모든 폴더는 Visual Code가 삭제 되어도 삭제 되지 않고 남아 있어서 다시 Visual Code 설치 하려고 하면 기존 설정 정보가 남아 있는 문제가 발생함
프로그램을 완전이 초기화고 재 설치 시 아래 폴더들을 삭제 바람

### 1. settings, keybindings, snippets 설정 파일 경로
```
%APPDATA%\Code\User
```

### 2.extensions 설정 정보 
```
%USERPROFILE%\.vscode\
```

### 3.elixir-ls 서버가 설치 되어있는 위치
```
%LOCALAPPDATA%\mix
```

### 4. 기타 설정파일 위치
```
%APPDATA%\Code
```
