
### Visual Code 실행 후 Ctrl+Shift-X 검색하여 설치 할 것

#### 1) 필수 Extensions
```
ElixirLS: Elixir support and debugger
Phoenix Framework
Simple Ruby ERB
Tailwind CSS IntelliSense
```


#### 2) 선택 사항 Extensions
```
vscodevim
```


### 3) 설치 파일 다운 받는 법
왼쪽 Bar 메뉴 버튼에 Extensions(Ctrl+Shift+X) 클릭 INSTALLED된 것 중 하나 선택 후  마우스 오른쪽 Download VSIX -> 다운 폴더 지정
