### 1. 특정 버전 설치

```
mix archive.install hex phx_new 1.7.21
```
mix archive.install hex phx_new 1.7.21  버전 번호를 입력 시 설치됨

### 2. 안정화 버전 설치
```
mix archive.install hex phx_new
```

### 3. 프로젝트 생성
```
mix phx.new 프로젝트명

## DB없이 간단히 테스트시 사용
mix --no-ecto phx.new 프로젝트명
```

### 4. UUID Schema키 생성
```
mix phx.new --binary-id 프로젝트명
```


### 5. UUID 키 생성 및 날짜 형식 설정(config.exs)
```
import Config

config :프로젝트명
  ecto_repos: ... 생략 ...
  generators: [timestamp_type: :local_datetime, binary_id: true]
  # timestamp_type : utc_datetime(영국시간), local_datetime(국내시간)
  # 스키마 키생성시 번호가 아니라 uuid를 사용시 binary_id: true 설정
```
